01 — Architecture
Conception objet
Un système extensible basé sur l'interface Domaine, le pattern Composite pour les catégories, et un parseur XML intégré.
🌳

Categorie

Nœud de l'arbre typologique. Possède un nom, une mère, des filles, et un dictionnaire de caractéristiques. Méthodes estCompatible() et validerHierarchie().

📐

Domaine (interface)

Contrat pour les domaines de valeurs. Deux méthodes : contient(valeur) et estInclusDans(domaineMère). Implémenté par Intervalle et Ensemble.

📏

Intervalle

Domaine numérique [inf, sup]. Contient un nombre si inf ≤ val ≤ sup. Inclusion vérifiée par comparaison des bornes.

📋

Ensemble

Domaine discret {e₁, e₂, …}. Contient une chaîne si elle est dans le Set. Inclusion vérifiée par containsAll().

👁️

Observation

Map<String, Object> associant chaque caractéristique à sa valeur observée. Sert d'entrée pour la classification.

🗂️

Typologie

Charge un fichier XML, construit l'arbre, valide la hiérarchie, et lance la classification récursive depuis la racine.

🔑 Contrainte d'inclusion

Chaque catégorie fille doit avoir ses domaines inclus dans ceux de sa mère. Exemple : si « arbre » a taille=[5,40], alors « épicéa » peut avoir taille=[20,30] mais pas taille=[20,50].

Categorie.java
public boolean estCompatible(Observation obs) {
    for (Caracteristique caracCat : this.caracteristiques.values()) {
        String intitule = caracCat.getIntitule();
        // L'observation doit posséder CHAQUE caractéristique requise
        if (!obs.aCaracteristique(intitule)) return false;
        Object valeurObs = obs.getValeur(intitule);
        Domaine domaineCat = caracCat.getDomaine();
        if (!domaineCat.contient(valeurObs)) return false;
    }
    return true;
}

public void validerHierarchie() throws ValidationException {
    if (this.mere != null && !this.nomMere.equals("TOP")) {
        for (Caracteristique caracMere : mere.getCaracteristiques().values()) {
            String intitule = caracMere.getIntitule();
            if (caracteristiques.containsKey(intitule)) {
                Domaine domaineFille = caracteristiques.get(intitule).getDomaine();
                // Le domaine fille DOIT être inclus dans le domaine mère
                if (!domaineFille.estInclusDans(caracMere.getDomaine()))
                    throw new ValidationException("Incohérence: '" + intitule + "'");
            }
        }
    }
    for (Categorie fille : filles) fille.validerHierarchie();
}
Domaine.java + implémentations
public interface Domaine {
    boolean contient(Object valeur);
    boolean estInclusDans(Domaine domaineMere);
}

public class Intervalle implements Domaine {
    private final double inf, sup;

    @Override
    public boolean contient(Object valeur) {
        if (!(valeur instanceof Number)) return false;
        double val = ((Number) valeur).doubleValue();
        return val >= inf && val <= sup;
    }

    @Override
    public boolean estInclusDans(Domaine m) {
        if (!(m instanceof Intervalle)) return false;
        Intervalle mere = (Intervalle) m;
        return mere.getInf() <= inf && mere.getSup() >= sup;
    }
}

public class Ensemble implements Domaine {
    private final Set<String> elements;

    @Override
    public boolean contient(Object v) {
        return (v instanceof String) && elements.contains((String) v);
    }

    @Override
    public boolean estInclusDans(Domaine m) {
        return (m instanceof Ensemble) && ((Ensemble) m).getElements().containsAll(elements);
    }
}
Typologie.java — classifier()
public List<Categorie> classifier(Observation obs) {
    List<Categorie> compatibles = new ArrayList<>();
    if (racine != null) classifierRecursif(racine, obs, compatibles);
    return compatibles;
}

private void classifierRecursif(Categorie cat, Observation obs,
                                 List<Categorie> resultats) {
    if (cat.estCompatible(obs)) {
        resultats.add(cat);
        // Descente récursive UNIQUEMENT si le parent match
        for (Categorie fille : cat.getFilles())
            classifierRecursif(fille, obs, resultats);
    }
    // Si le parent ne match pas, on ne teste pas ses enfants (élagage)
}
